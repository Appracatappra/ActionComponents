Detailed Description

The ActionComponents.ACImage is a static class that adds several helpful utilities for dealing with images as Bitmaps.

The ActionComponents.ACImage attempts to maintain the same calling structure and functionality with the iOS version of the ActionPack to assist in building cross platform mobile applications. Available in ActionPack Business or Enterprise only.
