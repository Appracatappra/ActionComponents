Detailed Description

The ActionComponents.ACTableViewDelegate handles all user interaction with the UITableView being controlled by it's parent ActionComponents.ACTableViewController. Events are processed and passed to the ActionComponents.ACTableSections and ActionComponents.ACTableItems contained within the ActionComponents.ACTableViewController's ActionComponents.ACTableViewDataSource. As a result you may never need to create a custom version of this class to support your UITableViews.

Available in ActionPack Business or Enterprise only.
