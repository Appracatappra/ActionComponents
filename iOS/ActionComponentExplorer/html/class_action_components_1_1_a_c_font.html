Acts as a replacement for the UIKit.UIView.StringSize method deprecated in iOS7. This function calculates the width and height of a single line of the specified string if it were rendered with the specified: font, width constraint, and line-break mode.

Returns
The height and width constrained to the given maximum sizes.
Parameters
textThe text to measure.
fontThe font that will be used to draw the string.
contrainedToThe maximum width and height.
breakModeThe line breaking mode.
