Detailed Description

The ActionComponents.ACWebView is a custom UIWebView with several helper methods that make it easier to use and require fewer lines of code to implement such as LoadFile or LoadURL. It has built in Dragable support that can be limited in movement by a ActionComponents.ACViewDragConstraint applied to the X and/or Y axis. It defines helper events for being Touched, Moved, and/or Released and can be set to automatically become the front view when it is touched. And provides methods to make moving, rotating, and resizing the ActionComponents.ACWebView easier with less code.

Available in ActionPack Business or Enterprise only.
