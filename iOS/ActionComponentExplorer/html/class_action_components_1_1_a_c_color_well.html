Detailed Description

The Action Color Well displays a framed color well that shows the currently selected color and allows the user to tap the well which raises the Touched event.

Property Documentation

bool ActionComponents.ACColorWell.AutoPresentHSBColorPicker
getset

Gets or sets a value indicating whether this T:ActionComponents.ACColorWell will automatically present a HSB Color Picker to select a new color.

true if auto present a HSB Color Picker; otherwise, false.

The ParentViewController property MUST be set to the UIViewController that the ACColorWell is hosted in before the HSB Color Picker can be automatically presented.
