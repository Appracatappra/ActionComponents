Detailed Description

A cross-platform tool for working with colors in the iOS and Android Mobile Operating systems. ACColor represents a union of nearly all of the options and features of both UIColor from iOS and the Color object from Android so the property or method used on one platform can be used on another without change in syntax. Several new features are supported such creating a color from or returning a color as a CSS/Web Hex String in the form #RRGGBB or #AARRGGBB. ACColor is fully compatible with UIColor and Color so it can be used anywhere those objects would be used and can be cast to and from those types implicitly.

ACColor includes all of the color presets from iOS and from Android along with a few, custom "Action" colors.
